## UnityAdvFinal2026 – Final Project

This repository contains the final project for the Unity Advanced course.  
The game is a challenge‑oriented experience with player control, enemies, buffs, and clear win/lose conditions.

### Core Gameplay

- Player character controlled via keyboard, mouse or gamepad.
- Enemies and hazards that create a meaningful challenge.
- Buff system (including immunity buff) that temporarily modifies player abilities.
- UI that shows player progress toward victory and failure.

### Implemented Unity Features

- **Mandatory features**
  - Player/object control using keyboard, mouse, and gamepad.(not compleete menu control on gamepad)
  - Animation system for the player and multiple scene objects.
  - At least three scenes with transitions managed by a scene loader/manager.
  - Game Over and Victory screens implemented via dedicated canvases.
  - Pause functionality, with an Options menu planned for both main menu and in‑game pause.

- **Core Unity tools**
  - Prefab variants (e.g., for enemies).
  - Coroutines (timers for enemy spawning and other delays).
  - Scriptable Objects (configuration for buffs and other data).
  - Unity/C# events (spawners, win conditions, UI updates).
  - Animator controllers for player and UI elements.
  - Transform‑based movement for the player and objects.
  - Navigation system (2D navmesh‑based navigation).
  - UI built with Unity UI.

- **Advanced tools**
  - Custom Inspector (grid map viewer).
  - Editor scripts (hierarchy painter).
  - Controller support (gamepad input).
  - Assembly Definitions for organizing code into multiple assemblies.
